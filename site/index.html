Dashboard Navigation

  • Intro: a brief explanation of dashboard features and adjustable settings
  • National View: dashboard showing the percentage of states that flipped in each election
  • State View: dashboard showing how often each state flipped in a set time window

Glossary

  • Flip: when a presidential candidate wins a state the opposing party won in the previous election
  • Flip Frequency: the percentage of window elections in which a particular state flipped
  • Flip Frequency Distribution: a chart showing how many states flipped at various frequencies
  • Rolling Average: a line in which each point averages the percentage of states flipped over a set number of previous elections
  • Time Window: a set of consecutive elections

Limitations and Caveats

  • Only flips between the Democratic and Republican parties are counted
  • Time ranges start at 1856, the first election in which a Republican candidate ran for president
  • D.C. is counted as a state
